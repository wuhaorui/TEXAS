# 2026-04-28 工作日志

## 1. 修复下注轮转逻辑（上午）

### 问题
用户报告：发牌前每个人下注一样多之后不发牌，每个人一直在点过牌。

### 修复方案
使用 `actedThisPhase` (Set) 跟踪本阶段已行动的玩家：
- 玩家行动 → 记录到 `actedThisPhase`
- 如果加注 → 清除其他玩家的已行动标记
- 一轮结束条件：`所有active玩家都已行动` 且 `下注金额相等`
