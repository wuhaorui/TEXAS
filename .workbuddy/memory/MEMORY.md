# MEMORY.md - 长期记忆

## 项目：德州扑克单人/多人练习网站

### 技术实现
- 纯HTML/CSS/JS单文件实现（单人版：index.html）
- Node.js + Socket.io 多人在线版本
- 绿色牌桌、木质边框、金色点缀的视觉风格
- 小盲10，大盲20

### 单人版功能
- 无需登录注册，输入昵称+选择筹码（1000/2000）即可开始
- 完整德州扑克流程：翻牌前→翻牌→转牌→河牌→摊牌
- 下注功能：弃牌、过牌、跟注、下注、加注、全下
- 快捷下注按钮：½ pot、pot、1½ pot、2x pot
- 键盘快捷键：F(弃牌)、C(过牌/跟注)、B(下注)、R(加注)、A(全下)
- 发牌动画：牌从牌桌右上角的牌叠位置飞出
- 切牌动画：发公共牌前显示切牌效果
- 牌桌显示一叠扑克（deck）和切掉的牌区域

### 多人在线版（server.js）
- 创建房间 → 生成分享链接 → 好友点击加入
- 2-8人同时在线对战
- 实时同步牌局状态
- 房主可开始游戏
- 完整德州扑克规则
- 自动判断胜负和牌型

### 文件结构
- index.html - 单人版
- public/index.html - 多人版前端
- server.js - Node.js + Socket.io 后端
- package.json - 依赖配置

### 踩坑经验
- 底池计算：bet/raise时先用 previousBet 记录原值，再计算 chipsToAdd，最后才更新 pot
- 公共牌时机：必须等下注轮结束后通过 advancePhase 才能发
- 发牌顺序：玩家→对手→玩家→对手交替发牌（一人一张交替）
- Socket.io callback一定要处理返回值
- Room对象属性：是 `room.id` 不是 `room.roomId`，后者是undefined
- 下注轮转判断：不能只用"轮回到第一个玩家"判断，需要用 actedThisPhase 集合跟踪已行动玩家
- 加注处理：有人加注后需重置其他玩家的已行动标记，让他们重新跟注
- 一轮结束条件：所有active玩家都已行动 且 下注金额相等
- Railway连接：需要添加心跳配置 pingTimeout/pingInterval 防止连接超时
